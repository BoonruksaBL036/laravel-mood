สำหรับผู้ใช้ใหม่

เทคโนโลยีที่ใช้

  • Backend: Laravel 12.0, PHP 8.2+
  • Frontend: Tailwind CSS
  • Database: MySQL/PostgreSQL/SQLite
  • Security: CSRF, Password Hashing

3. ระบบ Login

ฟีเจอร์หลัก

  • • ฟอร์มเข้าสู่ระบบ (Email/Password)
  • • ช่อง Remember Me
  • • ลิงก์ Forgot Password
  • • ปุ่ม Social Login

Controller Methods

  • • showLoginForm() - แสดงหน้าฟอร์ม
  • • login() - ประมวลผลเข้าสู่ระบบ
  • • logout() - ออกจากระบบ
  • • checkAuth() - ตรวจสอบสถานะ

การใช้งาน:

ไปที่ http://localhost:8000/login → กรอก Email/Password → คลิก Sign In

4. ระบบ Register

กฎการตรวจสอบข้อมูล

Name

  • • จำเป็นต้องกรอก
  • • 2-255 ตัวอักษร

Email

  • • รูปแบบ Email ถูกต้อง
  • • ไม่ซ้ำในระบบ

Password

  • • ขั้นต่ำ 8 ตัวอักษร
  • • มีตัวพิมพ์เล็ก/ใหญ่
  • • มีตัวเลขและสัญลักษณ์

การใช้งาน:

ไปที่ http://localhost:8000/register → กรอกข้อมูล → ยอมรับเงื่อนไข → สร้างบัญชี

7. การแก้ไขปัญหา

ไม่สามารถเข้าสู่ระบบได้

  • • ตรวจสอบ Email และ Password
  • • ลบ Cache: php artisan cache:clear
  • • ตรวจสอบการเชื่อมต่อฐานข้อมูล

หน้าเว็บไม่แสดงผล

  • • รัน: php artisan serve
  • • รัน: npm run dev
  • • ลบ View Cache: php artisan view:clear

คำสั่งที่มีประโยชน์

php artisan cache:clear # ล้าง Cache
php artisan migrate:status # ตรวจสอบ Migration
php artisan tinker # Laravel REPL
